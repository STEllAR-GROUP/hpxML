***********************
 AMR Support Libraries 
***********************

.. sectionauthor:: Bryce Lelbach 

Some of the HPX |amr|_ applications written by |matt|_ can optionally use
third-party libraries for and I/O (|rnpl|_). While these libraries are not
required, you will not get SDF output without the RNPL library.
